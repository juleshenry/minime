## Brainstorm Framework

NEW! The Brainstorm Framework is an AI-powered tool that:
- 🎯 **Project Manager Mode**: Analyzes your codebase and provides strategic direction
- 💡 **Idea Generator**: Produces K improvement ideas for your project
- 📝 **Issue Creator**: Automatically converts ideas into GitHub issues

Get started:
```bash
python3 brainstorm.py --num-ideas 5
```
